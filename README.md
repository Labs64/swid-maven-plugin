## Quick Start

The recommended way to get started using `swid-maven-plugin` in your project is with a plugin management system â€“ the Maven snippet below can be copied and pasted into your build.

Maven:
```xml
<plugins>
  <plugin>
    <groupId>com.labs64.mojo</groupId>
    <artifactId>swid-maven-plugin</artifactId>
    <version>0.1.0</version>
  </plugin>
```
